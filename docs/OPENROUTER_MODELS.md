## النماذج التي ليس لها تطبيق أو API مباشر للعميل (API Only)

جميع النماذج المذكورة أعلاه هي API فقط وليس لها تطبيق مباشر للعميل. يجب دعمها عبر OpenRouter API.

## النماذج التي لها تطبيق مباشر للعميل

لا توجد نماذج في OpenRouter لها تطبيق مباشر للعميل. جميع النماذج متاحة عبر API فقط.
